# Brugerroller

### Medarbejder

| Datatype | Beskrivelse |
| :--- | :--- |
| Aftaler | Kan læse og skrive \(priser undtaget\). |
| Kunder | Kan læse og skrive i relation til tildelte opgaver. |
| Opgaver | Kan læse og skrive \(priser undtaget\). |
